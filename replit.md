# prism-lang

A new programming language project.

## Project Structure

- `index.js` - Entry point: minimal Express web server serving a landing page on port 5000
- `package.json` - Node.js project configuration

## Tech Stack

- **Runtime:** Node.js 20
- **Framework:** Express 5
- **Port:** 5000 (webview)

## Running the App

```bash
npm run dev
```

The app is served at port 5000.
